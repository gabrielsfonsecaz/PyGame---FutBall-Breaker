PyGame - Futball Breaker - Projeto Final 1º semestre - Gabriel de Sousa Fonseca

INSTRUÇÕES:
1. Abra o VsCode
2. Abra a pasta 'BrickGame'
3. Execute o arquivo 'PyGame.py'
4. Pressione a tecla 'Espaço' na tela inicial
5. Controle a chuteira por meio das setinhas para evitar que a bolinha caia do cenário
6. Cada gol quebrado fornece 75 pontos
7. O jogador tem 3 vidas no total
8. Após perder as 3 vidas o jogo é finalizado e o score do jogador é guardado
